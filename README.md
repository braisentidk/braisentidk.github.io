## Edición
Pulsa **✏️ Editar**, modifica celdas y luego **💾 Guardar**. Se guarda en *localStorage* del navegador.

## Publicación
Sube la carpeta a un repo y activa **GitHub Pages → Deploy from a branch → main / (root)**.
